Selected Publications

  1. ICLR
    OBS-Diff: Accurate Pruning For Diffusion Models in One-Shot
    Junhan Zhu, Hesong Wang, Mingluo Su, Zefang Wang, and Huan Wang
    In International Conference on Learning Representations (ICLR), 2026
    Abs arXiv HTML Code

    We propose the first training-free, one-shot pruning framework for diffusion models, demonstrating broad applicability across diverse architectures and pruning granularities.

  2. CVPR
    Cross-Resolution Diffusion Models via Network Pruning
    Jiaxuan Ren*, Junhan Zhu*, and Huan Wang
    In IEEE/CVF Conference on Computer Vision and Pattern Recognition (CVPR) Findings, 2026
    Abs arXiv HTML

    We introduce CR-Diff, a framework that repurposes network pruning to enhance generalizability by removing adverse weights that cause degradation at non-default resolutions.

Selected Projects

  1. Project
    SparAlloc: A Simple and Modular Framework for Decoupled Sparsity Allocation in Layerwise Pruning for LLM
    Code

    A modular benchmark toolkit for decoupled sparsity allocation in LLM pruning. Enables fair comparison and flexible combination of pruning algorithms.

  2. Project
    Paper2Demo: Turn Any Academic PDF into a Stunning Demo Video — Fully Automatically
    Code

    An automated pipeline that converts research papers into ~30s animated video presentations. Extracts text and figures via PyMuPDF, structures content with LLM APIs, and renders polished videos with domain-aware theming using Remotion.
